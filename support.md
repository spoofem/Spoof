 # Support Guide

## Getting Help

### Support Tickets
1. Use "Create Ticket" in bot menu
2. Clearly describe your issue
3. Include relevant details
4. Wait for admin response

### Response Times
- Normal tickets: 24 hours
- Urgent issues: 1-6 hours
- Balance issues: ASAP

### Best Practices
- One ticket per issue
- Be clear and concise
- Include your user ID
- Provide screenshots if needed

## Common Issues

### Search Problems
- Check balance
- Verify BIN format
- Monitor daily limits
- Check for bans

### Purchase Issues
- Verify balance
- Check file availability
- Download promptly
- Save files locally

### Balance Issues
- Screenshot balance
- Note transaction times
- Keep payment proof
- Contact admin directly
